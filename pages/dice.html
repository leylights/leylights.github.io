This is an online dice roller, designed for use in a tabletop RPG setting. Of course, it does not need to be used as such, as it is simply a stylized random number generator.

To use this contraption, enter the following in the grey field above:

(The number of dice you would like to roll) + The letter "d" + (The number of sides on each die)

For example, rolling two six-sided dice would be entered as 2d6.

Alternatively, simply click on one of the dice at the bottom of the screen to roll a standardized amount.
