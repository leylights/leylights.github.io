How to use

The upper box is for the values of the dice. This follows tabletop game notation of <count>d<value>. That is, one six-sided die would be represented as 1d6, four six-sided dice would be represented as 4d6, and four twenty-sided dice would be represented as 4d20.

Different values of dice can be added or subtracted from each other, so combinations like '3d6 + 2d4' are valid.

The lower box is the number of 'experiments' that will take place. If the count is 100, then the given set of dice will be rolled 100 times, and their values will be recorded.

The more experiments you put in the lower box, the more accurate the dice distribution will be to its true set of probabilities.

Why does a bell curve appear so often?

This is a result of the Central Limit Theorem, a statistics theorem stating that if many random results are added together, they will form a bell curve.
